# MarkDown优点
1.纯文本内容，兼容所有文本编辑器和字处理软件
2.可以放到版本管理系统，追踪历史变更
3.轻松导出HTML，PDF和本身的.md文件
4.简洁、高效、可读、直观、学习成本低
5.专注你的文字内容而不是排版样式，安心写作

# 用途
- IT人事   写日志  技术文稿 记录代码片段  撰写文档
- 科研人员、学生   撰写科技论文  记笔记
- 求职者   制作求职简历  
- 其他  撰写博文  在线编辑文章等
